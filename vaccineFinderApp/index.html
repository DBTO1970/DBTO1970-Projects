COVID-19 Vaccine Administration and Case Data

Vaccines Given: *

This number represents the total number of vaccinations administered in Maryland. It includes First Dose, Second Dose, Fully Vaccinated totals for the State of Maryland. The Percent ( % ) is based on a total state population of 6.06 Million people.


A  Green  indicator shows an increase in doses administered from the previous day's data. A  Red   indicator shows a decrease, while a  Yellow  indicator shows no change from the previous data set.


*Updated via Maryland.gov COVIDLink

Cases: *(State-wide, per 100K people)

This number represents the total number of current COVID-19 cases in Maryland. The number is represented as a 7-day average per 100K.


A  Green   indicator shows a decrease in total current cases, state-wide, from the previous day's data. A   Red   indicator shows an increase, while a  Yellow   indicator shows no change from the previous data set.


*Updated via Maryland.gov COVIDLink
